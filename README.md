# Beyond the Efficient Frontier: Deep Reinforcement Learning for Portfolio Allocation

## Overview

This project investigates whether Deep Reinforcement Learning (DRL) can outperform traditional portfolio optimization methods in terms of risk-adjusted returns.

The study compares:

- Equal Weight Portfolio
- Markowitz Mean-Variance Portfolio
- Risk Parity Portfolio
- Black-Litterman Portfolio
- Deep Reinforcement Learning (PPO)

using historical stock market data from Yahoo Finance.

---

## Research Objective

To evaluate whether a Deep Reinforcement Learning agent can generate superior portfolio performance compared with classical portfolio optimization techniques.
